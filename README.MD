## Installation
Delilah requiere [Node.js](https://nodejs.org/) v12+ to run.
Instalar las dependencias y correr el servidor.:
```sh
git clone https://github.com/sanbega/api-node
cd delilah-resto
npm i
npm run start:dev
```
Para ingresar a la documentacion hecha en swagger: 
```sh
http://localhost:3000
```
